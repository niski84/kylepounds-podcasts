# Kyle Pounce Podcast Archive

Auto-generated audio episodes about Kyle Pounds' work, interweaving articles from kylepounds.com into 12-minute deep-dives via Google NotebookLM. Built by [site-patrol](https://kylepounds.org/patrol/).

## How it works

1. Every 7 days, site-patrol's auto-curator picks the longest-unused theme from a rotation (Bike Racing, Kayaking, Origin Stories, Family History, Ancient Timeline, Original Ideas).
2. It samples 3 random articles from that theme's path on the Kyle Pounce Library.
3. The article URLs are handed to NotebookLM with a Bill & Ted-style prompt — exuberant, optimistic, Kyle as the central visionary.
4. The mp3 gets re-encoded to 64kbps mono (~5MB), committed here, and surfaced on the dashboard + in the weekly newsletter.

## Layout

```
episodes/
  2026-06-01-1.mp3      ← audio
  2026-06-01-1.yml      ← sidecar: title, sources, generated_at
  ...
```
